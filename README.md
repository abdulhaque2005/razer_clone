# 🖤 Razer Website Clone  

## 🚀 Overview  
This project is a fully responsive clone of the Razer website, built using only HTML5 and CSS3. The main goal of this project is to practice real-world frontend development by recreating a modern gaming brand website without using JavaScript or any frameworks.  

The design focuses on dark theme UI, modern layout, and smooth responsiveness across all devices including mobile, tablet, and desktop screens. It demonstrates how premium-looking websites can be built using only core web technologies.  

---

## 🎯 Features  
- Fully responsive design  
- Built using only HTML & CSS  
- Dark theme modern UI  
- Real-world website clone  
- Mobile, tablet, and desktop friendly  
- Flexbox-based layout and positioning
